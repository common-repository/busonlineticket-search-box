== Description ==

The BusOnlineTicket Search Box by BusOnlineTicket.com offers users a versatile search box widget and shortcode for displaying search boxes on their WordPress websites. This search box enables users to search for bus, train, and ferry tickets in Malaysia and Singapore. Users can customize the appearance of the search box, save and edit data, and enjoy various features to enhance their search box experience. This plugin is specifically designed for affiliates of BusOnlineTicket.com.

== Features ==

1. **Save and Edit Data:** Save and edit data entered into the search box for a personalized experience.

2. **Resizable Search Box:** Choose from multiple sizes of the search box to fit your website design.

3. **Preview Functionality:** Preview how the search box will look on your website before implementing it.

4. **CSS Editor:** Customize the appearance of the search box in detail with the built-in CSS editor.

5. **Direct Shortcode Copy:** Easily copy the generated shortcode for quick implementation on your pages or posts.

== Installation ==

1. From your Wordpress dashboard, visit 'Plugins' -> 'Add New'.
2. Search for "BusOnlineTicket Search Box".
3. Install and activate the plugin through the 'Plugins' menu in WordPress.
4. Configure the plugin settings in the WordPress dashboard under "BusOnlineTicket".

== Usage ==

1. Register as an affiliate with BusOnlineTicket.com to obtain your affiliate credentials.
2. Install and activate the plugin.
3. Configure your affiliate settings in the plugin.
4. Use the provided shortcode to embed the search box on your website.

== Third-Party Service Dependency ==

This plugin relies on a third-party service for certain functionalities. The included external script, [BusOnlineTicket JavaScript](https://cdn.busonlineticket.com/js/all_route.js), provides a comprehensive list of routes for bus, train, and ferry transportation in Malaysia and Singapore. The script enhances the search box functionality by dynamically populating route options, ensuring accurate and up-to-date information for users.
